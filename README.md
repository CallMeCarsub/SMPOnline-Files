This is a place where you can download the latest versions of mods for playing on SMPOnline.

# Ok how do I download it?
Go to the [page for the downloads](https://github.com/CallMeCarsub/SMPOnline-Mods/releases/tag/latest).

Once you're there, you may need to expand "Assets". The files your looking for are the .jar files.

If you're still lost, you should be using the [official Modrinth modpack](https://modrinth.com/project/smponline-s5-pack). Don't bother me if you can't figure out GitHub releases, please.
